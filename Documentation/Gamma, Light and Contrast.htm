Gamma, Light and Contrast

These filters alter color intensity on various parts of an image. They commonly used to produce effects such as "warm", "cold", "dark" image and so on.
To best understand these filters have a look at bellow examples.

Gamma

To use gamma filter on all selected images:

  • Through main menu, click Filters → Gamma → Increase / Decrease
    - Or -
    Press Shift + 1 keyboard shortcut to increase or Ctrl + 1 to decrease.
    - Or -
    Use the right-click menu on the bottom workspace area.

Light

To use light filter on all selected images:

  • Through main menu, click Filters → Light → Increase / Decrease
    - Or -
    Press Shift + 2 keyboard shortcut to increase or Ctrl + 2 to decrease.
    - Or -
    Use the right-click menu on the bottom workspace area.
Original image. With increased filter level. With decreased filter level.

Contrast

To use light filter on all selected images:

  • Through main menu, click Filters → Contrast → Increase / Decrease
    - Or -
    Press Shift + 3 keyboard shortcut to increase or Ctrl + 3 to decrease.
    - Or -
    Use the right-click menu on the bottom workspace area.

Notes

  1. Artefacts present in compressed images such as Jpeg will be more visible with increased level of these filters. This is because Jpeg compression relies on optical illusions which are more easily hidden on darker images.

1. Colorize Filter

This filter will alter image colors while preserving brightness and shadows. It is usually used to add color to grayscale images. IconMasterXP allows you only to colorize entire image with one single color. In icon design this filter is very popular as it allows you to create multiple color variations of the same grayscale icon.

  • Through main menu, click Filters → Colorize
    - Or -
    Use the right-click menu on the bottom workspace area.

A dialog will show up allowing you to chose color parameters, see bellow.

You can either combine Hue and Saturation sliders to choose a desired color or you can simply click Color button. Strength of this effect depends on the Blend slider (left side: weakest, right side: strongest).

Bellow you can see an example of this effect.

Notes

  1. To convert a colored image into a grayscale image: leave all the parameters as default but drag Blend to the maximum.

2. Negative Filter

This is a very simple filter — it inverts color of every pixel on the image. Black will become white, blue will become yellow and so on. To use this filter:

  • Through main menu, click Filters → Negative
    - Or -
    Use the right-click menu on the bottom workspace area.

1. Image Transparency

Transparent color is a color that will be hidden when image is displayed, that is, software that displays the image will replace pixels that use transparent color with their own background pixels.
Only Png and Ico formats support transparency. With 32bit images you should use alpha channels (transparency masks) instead of selecting a transparent color.
See Color Modes for more information on image bit-depth.

To pick a transparent color for selected image:

  • Press the Transparent button from the toolbar.
    - Or -
    Through main menu, click Edit → Transparent Color → Pick From Image
    - Or -
    Use the right-click menu on the bottom workspace area.
  • Pick a color directly from the image with your mouse. You can zoom in an image for better convenience.

To manually set RGB values for a transparent color:

  • Through main menu, click Edit → Transparent Color → Write RGB Values
    - Or -
    Use the right-click menu on the bottom workspace area.
  • Write desired red, green and blue values or use HSB values instead.

To remove a transparent color (revert to opaque image):

  • Through main menu, click Edit → Transparent Color → Remove
    - Or -
    Use the right-click menu on the bottom workspace area.

2. Replacing Colors

To replace one color with another:

  • Through main menu, click Edit → Replace Colors
    - Or -
    Use the right-click menu on the bottom workspace area.
  • A dialog will show up, select color to be replaced.
  • Now select replacement color.

There is no special color comparison algorithm here, only colors that are identical with your specifications will be replaced.

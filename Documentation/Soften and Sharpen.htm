Soften

Soften filter alters color differences in an image through Gaussian blur filter. To put it simply, it basically blurs the original image. It will greatly affect edges in an image, without loosing important detail.
It is mostly used to create shadow effects or smooth and simple animations with couple of frames.
To use soften:

  • Through main menu, click Filters → Soften
    - Or -
    Press Ctrl + T keyboard shortcut.
    - Or -
    Use the right-click menu on the bottom workspace area.

After this, all selected images will have the filter applied. Bellow is an example of this filter.

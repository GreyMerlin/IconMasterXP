Exporting Image Files

Use export feature to convert previosuly imported images into another format. To export images:

  • Press the Export button on the toolbar.
    - Or -
    Through main menu: File → Export
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Ctrl + S keyboard shortcut.

After that, a dialog will show up allowing you to choose a desired image format.

Then, you will be asked for a desired export location on your file system. You can enter a full path into the Folder text field and press Enter.

After that, all the images (not only selected ones) you have previously imported regardless of their original format will be exported into a chosen format.

Multipage Export

IconMasterXP also supports exporting so-called multipage images. Those images are files containing more than one image (multiple pages). For now, only Icon (ICO) format supports this. You should use this feature if you want your icons to support multiple sizes and/or different colour modes.

To enable multipage feature:

  • Press the Multipage button on the toolbar.
    - Or -
    Through main menu: File → Multi-page
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Ctrl + P keyboard shortcut.

You will be asked to enter location and name of the file to export. All of the images will be contained in this one file. Remember that for now only Icon format is supported for this feature.

You can check export results by importing the image you just exported.

Jpeg Compression Setting

When using JPG format in export operations you can change jpeg compression settings. To do this:

  • Through main menu: File → Jpeg Settings
    - Or -
    Use the right-click menu on the bottom workspace area.

After that, all future exported jpeg files will be saved with chosen preferences. Your setting is saved to registry so it will not reset after you restart the program.

Duplicating Images

Use duplicate feature to create carbon-copy of each currently selected image.
To duplicate selected images:

  • Through main menu: Edit → Duplicate
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Ctrl + D keyboard shortcut.

Since duplicating is commonly used to change icon size or bit-depth, they will have same names as their parents. You can also use Duplicate to make backup copies of your images; you can later delete those copies when they are not needed anymore.

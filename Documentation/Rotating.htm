Rotating Images

You can rotate selected images to a desirable angle by using this feature. To rotate one or more images:

  • Through main menu: Edit → Rotate
    - Or -
    Use the right-click menu on the bottom workspace area.

After that, a dialog will show up allowing you to chose rotation angle and interpolation method (antialiasing).

You can also chose not to resize the image when rotating. Certain images cannot fit their canvas after being rotated so program will automatically resize it by default.

You can choose between five interpolation methods, each giving slightly different results. There is very small difference between these methods so you are encouraged to experiment. Most commonly used methods are bilinear for vector/text/line art and bicubic for photography - different results are given for different graphics.

Mirroring and Flipping

You can also quickly flip an image vertically or horizontally (mirror).

To mirror an image:

  • Through main menu: Edit → Mirror
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Ctrl + M keyboard shortcut.

To flip an image:

  • Through main menu: Edit → Flip
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Ctrl + F keyboard shortcut.

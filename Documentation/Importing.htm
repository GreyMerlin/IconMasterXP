Importing Image Files

Use Import feature to add images into working area for editing and later conversion. To import images into IconMasterXP:

  • Drag desired image files from Windows® Explorer or similar program into the workspace.
    - Or -
    Press the Import button from the toolbar.
    - Or -
    Through main menu: File → Import
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Ctrl + O keyboard shortcut.

After that, a dialog will show up providing you with standard explorer-like file browser to select one or more files.

If you already have some images imported, you will be asked if you want to remove them or not. You do not have to remove them, it does not matter how many times you imported images, they can just be added to the current list.

If you are importing PSD files, the import process may take few seconds to finish due the fact that IconMasterXP does not internally support PSD format. If an error is encountered with some file or files, IconMasterXP will report the error and continue importing rest of the files.

When program has imported all the files, you will get a sorted list of all valid images. You may see an exclamation mark on some icons in the list, this is due to an error while displaying preview image - it does not always mean that associated file is corrupt.

Notes

  1. Currently, GIF file format is not listed as allowed format. However, that format can be imported. You can import any GIF file, including GIF animations. However, you cannot export them.
  2. Some icons (and gifs) are multipage files, IconMasterXP will properly import all of the pages/images. See Multipage Feature for more information on multipage files.

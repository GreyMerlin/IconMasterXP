Undoing and Reverting Operations

With Undo command, you can undo one or more changes done to image or images. IconMasterXP records all changes done, so you can roll them back if you are not satisfied with results.
To undo:

  • Press the Undo button on the toolbar.
    - Or -
    Through main menu: Edit → Undo
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Ctrl + Z keyboard shortcut.

Some changes cannot be undone, including:

  • Import/Export operations
  • Changes made in JPEG Settings dialog
  • Item removal
  • Item duplication

Reverting Images

You can also revert an image back to its original state when it was imported. To do this:

  • Through main menu: Edit → Revert to Original
    - Or -
    Use the right-click menu on the bottom workspace area.

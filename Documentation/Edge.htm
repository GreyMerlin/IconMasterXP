Edge Filter

Edge detector is usually used in image analysis but it can be also useful in design processes. To use edge filter:

  • Through main menu, click Filters → Edge
    - Or -
    Press Ctrl + E keyboard shortcut.
    - Or -
    Use the right-click menu on the bottom workspace area.

After this, all selected images will have the filter applied. Bellow is an example of this filter.

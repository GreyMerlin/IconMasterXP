Overview

IconMasterXP 4.9 is a general-purpose batch image processing tool aimed at professional designers and developers. Its main purpose to date is creation of fully qualified Windows® icons - now it works with 12 other image formats with the same level of quality. It has numerous features worth trying out if you are looking for a batch image processor with an elegant native Windows interface.

Process of making icons is not so simple, that is why most graphic processors exclude this format from their engine. That is an area in which IconMasterXP is the best, just plain – making icons. All you needs is an image created by any image processing program, IconMasterXP, and few button clicks to receive a fully compatible icon. There you have it, making icons in its simplest form.

Microsoft® Icon is most common graphic format used in Windows®, whatever you do – you will see icons – they give life to your desktop, professionalism to your software and charm to your design. Be sure to use the best tool for making icons.

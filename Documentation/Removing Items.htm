Removing Images

To remove currently selected images:

  • Press the Delete button on the toolbar.
    - Or -
    Through main menu: Edit → Remove Items
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Del keyboard shortcut.

This action cannot be undone and there is no warning to confirm deletion, double-check if you really want to remove selected items. Actual files associated with removed images will not be deleted from the hard drive.

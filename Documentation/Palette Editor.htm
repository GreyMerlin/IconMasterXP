Using Palette Editor

Palette editor allows you to change palette of a selected image. To edit palette colours:

  • Select a desired image – this feature cannot be used when multiple images are selected.
  • Through main menu: Edit → Palette

After that, a dialog will show up allowing you to edit image palette.

To change a particular colour: double-click on it.

Notes

  1. Only some image colour formats (bit-depths) support palettes, e.g. 32bit, 24bit and 1bit formats do not.

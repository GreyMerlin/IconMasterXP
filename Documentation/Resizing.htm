Resizing Images

You can resample selected images to a desirable size by using this feature. To resize one or more images:

  • Press the Resample button on the toolbar.
    - Or -
    Through main menu: Edit → Resize
    - Or -
    Use the right-click menu on the bottom workspace area.
    - Or -
    Press Ctrl + L keyboard shortcut.

After that, a dialog will show up allowing you to chose image size and interpolation method (antialiasing). There is very small difference between these methods so you are encouraged to experiment. Most commonly used methods are bilinear for vector/text/line art and bicubic for photography - different results are given for different graphics.

You can choose between five interpolation methods, each giving slightly different results.

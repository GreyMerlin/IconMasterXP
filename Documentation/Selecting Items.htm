Selecting Images

There are many ways to select multiple items in the list:

  • Use your mouse to click at empty space in the list, then if you drag the mouse you will see a gray selection rectangle which enables you to select desired items.
  • If you click on items while holding Ctrl key, your previous selection will not be lost after clicking on another item - that way you can select multiple items.
  • Hold shift button to select a range of items, e.g. click first and then last item if you want to select all items. This method applies only to items with same color depth.

To select all items:

  • Through main menu: Edit → Select All
    - Or -
    Press Ctrl + A keyboard shortcut.

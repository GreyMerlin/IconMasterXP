Threshold Filter

This filter is used to convert color images to monochromatic images by removing pixels based on their color and brightness. To use this filter:

  • Through main menu, click Filters → Threshold → Interactive / 50%
    You can use either interactive mode or just set the threshold to 50% value. Interactive mode is recommended as it will render different levels of effect on selected image in real time in the preview pane.
    - Or -
    Use the right-click menu on the bottom workspace area.
  • If you used interactive mode, press any key to stop it when you are satisfied with results. When interactive mode reaches maximum value it will start going backwards so take your time to choose an optimal version.

After this, same strength of the filter will be applied to all selected images. You can see some examples bellow.
